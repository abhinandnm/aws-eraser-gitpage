Official web and documentation repository for **AWS Eraser** — the fastest solution to scan, audit, and clean active billing resources across all AWS regions.

---

## 🚀 Quick Start (Local Viewing)

To run the documentation locally:

```bash
pip install -r requirements.txt
mkdocs serve
```

Open `http://127.0.0.1:8000/` in your browser.
